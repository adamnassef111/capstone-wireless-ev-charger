# Wireless EV Charging Protection and Monitoring System

## Overview
This project involved the design, simulation, and testing of a protection and monitoring system for a renewable-powered wireless EV charging setup. The system ensures safe operation by detecting fault conditions and disconnecting the power path when required.

The platform was designed to operate with variable input from renewable sources such as solar and wind, and to maintain safe electrical conditions across the wireless power transfer stage.

---

## My Contributions
- Designed analog protection circuits for over-voltage, under-voltage, and over-temperature conditions
- Developed comparator-based fault detection using LM311
- Built and tested circuits on breadboard and PCB
- Integrated relay-based isolation to disconnect the load during faults
- Supported system-level testing and troubleshooting
- Assisted in validating system performance under different operating conditions

---

## System Functionality
- Monitors voltage, current, and temperature conditions
- Detects abnormal operating conditions
- Automatically disconnects the power path using relays
- Provides a stable operating window (e.g., 20–29 V range during testing)
